Player Controller - управление передвижением игрока (также присутвует логика приседания
и фиксации к выступающей поверхности).

Player Sound Manager - управление звуком в зависимости от состояния игрока.

View Controller - управление видом с камеры + повороты модели игрока (можно ограничить 
угол обзора).

Cube In Head, Player Pack, Input Settings, Restart Area - классы созданные для работы
совместно с Game Manager (в основном только хранят какие-то данные).

Game Manager - управляет состоянием игры, а также содержит ссылки на общие для всех 
скриптов данные ( н.р. настройки ввода с клавиатуры ).
